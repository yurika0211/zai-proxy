# Skill 工具测试

此文件用于测试 Skill 工具。

## 可用技能列表

### 通用技能
- **update-config** - 配置 Claude Code 设置
- **keybindings-help** - 自定义键盘快捷键
- **simplify** - 审查并改进代码
- **loop** - 定期运行提示或命令
- **schedule** - 创建和管理定时远程代理

### API 集成
- **claude-api** - 构建 Claude API / Anthropic SDK 应用
- **bangumi-api** - 交互 Bangumi API（动漫/漫画/游戏）
- **blog_api** - 本地博客 API 操作与集成

### 可视化
- **excalidraw-diagram** - 生成 Excalidraw 图表
- **mermaid-visualizer** - 转换为 Mermaid 图表
- **json-canvas** - 创建 JSON Canvas 文件

### Obsidian 集成
- **obsidian-bases** - 创建 Obsidian Bases
- **obsidian-canvas-creator** - 创建 Obsidian Canvas
- **obsidian-cli** - 使用 Obsidian CLI
- **obsidian-markdown** - 创建 Obsidian Markdown
- **obsidian-homework** - 撰写课程作业
- **obsidian-mistake-book** - 创建错题本

### 其他
- **defuddle** - 从网页提取干净内容
- **git-version-control** - Git 版本控制专家
- **pdf-to-tutorial** - PDF 转化为教程笔记
- **security-best-practices** - 安全最佳实践审查
