  可用技能 (Skills)

  - update-config - 配置 Claude Code 设置
  - keybindings-help - 自定义键盘快捷键
  - simplify - 审查并改进代码
  - loop - 定期运行提示或命令
  - schedule - 创建和管理定时远程代理
  - claude-api - 构建 Claude API / Anthropic SDK 应用
  - bangumi-api - 交互 Bangumi API（动漫/漫画/游戏）
  - blog_api - 本地博客 API 操作与集成
  - defuddle - 从网页提取干净的 Markdown 内容
  - excalidraw-diagram - 生成 Excalidraw 图表
  - git-version-control - Git 版本控制专家
  - json-canvas - 创建和编辑 JSON Canvas 文件
  - mermaid-visualizer - 将文本转换为 Mermaid 图表
  - obsidian-bases - 创建和编辑 Obsidian Bases
  - obsidian-canvas-creator - 创建 Obsidian Canvas 文件
  - obsidian-cli - 使用 Obsidian CLI 交互
  - obsidian-homework - 在 Obsidian 中撰写课程作业
  - obsidian-markdown - 创建和编辑 Obsidian Markdown
  - obsidian-mistake-book - 创建和管理 Obsidian 错题本
  - obsidian-vault-structure - Obsidian Vault 结构
  - pdf-to-tutorial - 将 PDF 转化为 Obsidian 教程笔记
  - security-best-practices - 安全最佳实践审查
